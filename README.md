## 反射机制: Mirror
[Reflect.swift](https://github.com/monkey19911021/TEstSwift/blob/master/TEstSwift/Reflect.swift)
```
object 是 Struct
object 的类名: Person
Person 的属性个数是：2
第0个属性名: name，值: 菲妮
第1个属性名: age，值: 18


object 是 Class
object 的类名: Fruit
Fruit 的属性个数是：2
第0个属性名: name，值: 苹果
第1个属性名: count，值: 10


▿ TEstSwift.Fruit #0
  - name: 苹果
  - count: 10
```
